Modify app/layout.tsx to subset Geist fonts and preload key weights:

1. Update GeistSans import configuration (replace the import usage, not the import statement):
   ```typescript
   import { GeistMono } from "geist/font/mono";
   import { GeistSans } from "geist/font/sans";

   const geistSans = GeistSans({
     subsets: ['latin'],
     weight: ['400', '500', '600'],
     preload: true,
     display: 'swap',
     variable: '--font-geist-sans',
   });

   const geistMono = GeistMono({
     subsets: ['latin'],
     weight: ['400', '500'],
     preload: true,
     display: 'swap',
     variable: '--font-geist-mono',
   });
   ```

2. Update the HTML className to use the configured fonts:
   ```typescript
   <html
     className={`${geistSans.variable} ${geistMono.variable}`}
     // ... rest of props
   ```

- 500: Medium weight (font-medium)
- 600: Semibold (font-semibold)
- Mono needs 400, 500 for code blocks

If build warnings appear about missing font-bold (700), add it to the weight array. Per research, 400/500/600 covers most use cases.
