Install husky and initialize git hooks:
```bash

```

Create .husky/pre-commit hook to run Biome on staged files only:
```bash
#!/usr/bin/env sh
npx @biomejs/biome check --write --staged --no-errors-on-unmatched --files-ignore-unknown=true
```

Flags explained (from research):
- --staged: Only check files in git index (requires Biome 1.7.0+, project has 2.3.10)
- --write: Auto-fix issues and re-stage files (Biome 2.0+ handles re-staging automatically)
- --no-errors-on-unmatched: Don't fail if no files match
- --files-ignore-unknown=true: Gracefully handle file types Biome doesn't support

Make hook executable:
```bash
chmod +x .husky/pre-commit
```

Test hook by creating a temporary lint error and attempting to commit (hook should block or auto-fix).
